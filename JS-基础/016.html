看看匹配字符的实例

（1）匹配16进制颜色值

分析：

表示一个16进制字符，可以用字符组[0-9a-fA-F]。

其中字符可以出现3或6次，需要是用量词和分支结构。

使用分支结构时，需要注意顺序。（因为其惰性匹配的原理）

var regex = /#([0-9a-fA-F]{6}|[0-9a-fA-F]{3})/g
1

（2）匹配时间

共4位数字，第一位数字可以为[0-2]。

当第1位为2时，第2位可以为[0-3]，其他情况时，第2位为[0-9]。

第3位数字为[0-5]，第4位为[0-9]

var regex = /^[01][0-9]|2[0-3]:[0-5][0-9]$/
1

（3）验证邮箱

var regex = /^([a-zA-Z0-9_\-])+@([a-zA-Z0-9_\-])+(\.[a-zA-Z0-9_\-])+/
1

（4）验证身份证号码

身份证号码可能为15位或18位，15位为全数字，18位中前17位为数字，最后一位为数字或者X

var regex =  /(^\d{15}$)|(^\d{17}(\d|X|x)$)/
1

（5）验证电话号码

var regex = /^1[34578]\d{9}$/
1

其中(?!\b)怎么理解呢？

要求当前是一个位置，但不是\b前面的位置，其实(?!\b)说的就是\B。

因此最终正则变成了：/\B(?=(\d{3})+\b)/g。

（7）验证密码问题

密码长度6-12位，由数字、小写字母和大写字母组成，但必须至少包括2种字符。

要求必须包含数字可以用 (?=*[0-9]) 来表示。

就变成：

var reg = /(?=.*[0-9])^[0-9A-Za-z]{6,12}$/
1

现在要求必须包含两种，比如数字和小写字母，可以用(?=.*[0-9])(?=.*[a-z])

最终答案：

var reg = /((?=.*[0-9])(?=.*[a-z])|(?=.*[0-9])(?=.*[A-Z])|(?=.*[a-z])(?=.*[A-Z]))^[0-9A-Za-z]{6,12}$/
1

TIP

重点理解 (?=.*[0-9])^，拆分成(?=.*[0-9])和^。

先理解(?=.*[0-9])，表示该位置后面的字符匹配.*[0-9]，即，有任何多个任意字符，后面再跟个数字。再理解^，表示开头前面还有个位置（当然也是开头，即同一个位置，想想之前的空字符类比）。

另一种解法

“至少包含两种字符”的意思就是说，不能全部都是数字，也不能全部都是小写字母，也不能全部都是大写字母。

那么要求“不能全部都是数字”，怎么做呢？(?!p)出马！

对应的正则是：

var reg = /(?!^[0-9]{6,12}$)^[0-9A-Za-z]{6,12}$/
1

三种“都不能”呢？

最终答案是：

var reg = /(?!^[0-9]{6,12}$)(?!^[a-z]{6,12}$)(?!^[A-Z]{6,12}$)^[0-9A-Za-z]{6,12}$/
1

（8）获取 url 参数

function getUrlParam(url, key) {
	const arr = {}
	const regex = /\??(\w+)=(\w+)&?/g
    url.replace(regex, function(match, matchKey, matchValue) {
    	if (!arr[matchKey]) {
           arr[matchKey] = matchValue;
       } else {
       	   // 若存在同名参数，返回数组	
           var temp = arr[matchKey];
           arr[matchKey] = [].concat(temp, matchValue);
       }
    })
    
    return arr.hasOwnProperty(key) ? arr[key] : ''
}

（14） html转义和反转义

转义：

function escapeHtml(str) {
	const escapeChars = {
		'<': 'lt',
		'>': 'gt',
		'""': 'quot',
		'&': amp,
		'\': '#39'
	}
	return str.replace(new RegExp(`[${Object.keys(escapeChars).join('')}]`, g), function(match) {
		return `&${escapeChars[match]};`
	})
}

反转义：

就是转义的逆过程

function unescapeHtml(str) {
    const htmlEntities = {
        'lt': '<',
        'gt': '>',
        'quot': '""',
        'amp': '&',
        '#39': '\'
    }
    return str.replace(/\&([^;]+);/g, function(match, key) {
        if (htmlEntities[key]) {
            return htmlEntities[key]
        }
        return match
    })
}
